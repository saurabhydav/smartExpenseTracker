## Quick Start

### 1. Start Database
```bash
docker-compose up -d mysql
```

### 2. Run Backend
```bash
cd backend
./mvnw spring-boot:run
```

### 3. Run Mobile App
```bash
cd mobile
npm install
npm run android  # or npm run ios
```
